完整顺利的卸载node（避免踩坑！）

Code is never die！

1.卸载 node 本身、删除 nodejs 目录

提示：可以在系统控制面板–>所有控制面板项–>程序和功能 卸载 Node.js

2.手动删除C:\Program Files\nodejs\node_modules(如果你的文件是在这里的话)

3.手动删除C:\users\你的用户名\node_modules(如果你的文件是在这里的话)

这样就能完整卸载电脑上面的 node，然后也可以进行新版本下载
附官网地址：下载新版 nodejs
