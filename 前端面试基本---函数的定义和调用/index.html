1.0 函数的定义方式

  • 方式 1： 函数声明方式 function 关键字 (命名函数)

  • 方式 2： 函数表达式(匿名函数)

  • 方式 3： new Function() （函数也是对象，所以可以 new）（了解）

    注意：

    • Function 里面参数都必须是字符串格式
    • 第三种方式执行效率低，也不方便书写，因此较少使用
    • 所有函数都是 Function 的实例(对象)
    • 函数也属于对象
